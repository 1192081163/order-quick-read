# Code of Conduct

This project expects respectful, practical collaboration.

## Expected Behavior

- Be direct and constructive.
- Focus criticism on code, behavior, and tradeoffs.
- Assume good intent while still asking for evidence.
- Keep discussions welcoming to contributors with different backgrounds and
  experience levels.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Dismissing security concerns without investigation.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate
this code of conduct. Repeated or severe violations may lead to being blocked
from participating in the project.
